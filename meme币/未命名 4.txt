# 📡 MemeRadar - Meme币实时追踪平台

## 功能特性

### 🔴 实时信息流
- 实时抓取 X (Twitter) 上与 Meme 币相关的推文
- 自动识别推文中提及的代币标签 ($PEPE, $WIF 等)
- 高亮显示 KOL 推文、新代币发现、鲸鱼动向

### 📊 趋势分析
- 代币热度排行榜（实时更新）
- 迷你趋势图（Sparkline）
- 情绪分析评分 (Sentiment Score)
- 话题热力图 / 词云

### 🔔 智能提醒系统
- **新代币发现**: 首次检测到未知代币时提醒
- **KOL 喊单**: 知名 KOL 提及某代币时提醒
- **热度激增**: 提及量短时间内异常增长时提醒
- **鲸鱼关注**: 大户钱包关注某代币时提醒
- 声音提醒 + Toast 弹窗通知

### 👑 KOL 追踪
- 预设多个加密货币领域 KOL
- 追踪其最新推文和提及的代币
- 影响力评分

### ⚙️ 可配置
- X API Token 配置
- 区块链 RPC 配置
- 自定义追踪关键词
- 扫描频率设置
- 提醒规则自定义
